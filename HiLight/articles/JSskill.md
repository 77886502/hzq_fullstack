# JavaScript的小技巧

## 声明多个变量
```js
// 常规
let a;
let b = 2;

// 简写
let a,b=2;
```

## 为多个变量赋值
```js
// 常规
let a=1,b=2,c=3;

// 简写
let [a,b,c] = [1,2,3]
```

## 恰当的使用三元运算符
```js
// 常规
let a = 10;
let result;
if(a<100)
    result = true;
else
    result = false;

// 简写
let result = a<100?true:false;
```
